# Cypress End-to-End Testing

## Overview

This project contains automated end-to-end tests built with Cypress.

## Features

- Automated browser testing
- UI validation
- Regression testing
- Fast test execution

## Technologies

- Cypress
- JavaScript
- Node.js

## Installation

```bash
npm install
```

## Run Tests

Open Cypress

```bash
npm run cypress:open
```

Run headlessly

```bash
npm run cypress:run
```

## Project Structure

```
cypress/
  e2e/
  fixtures/
  support/

cypress.config.js

```
